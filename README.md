# LCD CONNECTIONS to a Raspberry Pi

|LCD Pin|	Function |	Pi Pin |
|------|------|------|
|1|	GND|	GND|
|2|	+5V	|+5V|
|3|	Vo (Contrast)|	GND|
|4|	RS	|GPIO-14|
|5|	RW	|GND|
|6|	Enable|	GPIO-18|
|7|	D0	|Not in Use|
|8|	D1	|Not in Use|
|9|	D2	|Not in Use|
|10|	D3|	Not in Use|
|11	|D4	|GPIO-24|
|12	|D5	|GPIO-23|
|13|	D6|	GPIO-8|
|14	|D7	|GPIO-25|
|15|	+5V (Backlight)|	+5V|
|16	|GND(Backlight)|	GND|
